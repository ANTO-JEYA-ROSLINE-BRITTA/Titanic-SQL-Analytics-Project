# Titanic SQL Analytics Project

## Overview

The Titanic SQL Analytics Project is a data analysis project that demonstrates the practical application of SQL for querying, analyzing, and extracting insights from structured data. Using the Titanic dataset, various SQL operations were performed to explore passenger demographics, survival patterns, fare distributions, and travel class information.

This project was developed as part of a Data Analytics Internship to strengthen database querying skills and gain hands-on experience with relational database management systems.

---

## Objectives

* Import and manage structured data in MySQL.
* Perform data retrieval and filtering using SQL queries.
* Analyze passenger information using aggregate functions.
* Apply sorting and grouping techniques for data exploration.
* Implement subqueries for advanced data analysis.
* Demonstrate JOIN operations between related tables.
* Create reusable database views for reporting purposes.
* Improve query performance using indexes.

---

## Tools and Technologies

* **Database:** MySQL
* **Interface:** MySQL Workbench
* **Dataset:** Titanic Dataset (train.csv)
* **Language:** SQL

---

## Dataset Description

The dataset contains passenger information from the Titanic disaster, including:

* Passenger ID
* Survival Status
* Passenger Class
* Name
* Gender
* Age
* Number of Siblings/Spouses Aboard
* Number of Parents/Children Aboard
* Ticket Information
* Fare
* Cabin Details
* Port of Embarkation

## Analysis Performed

### Gender-Based Survival Analysis

Analyzed passenger counts and survival statistics based on gender using grouping and aggregation techniques.

### Fare Analysis

Identified passengers paying fares above the average ticket price through subqueries and comparative analysis.

### Passenger Class Analysis

Examined passenger distribution across travel classes and explored class-based characteristics using JOIN operations.

### Statistical Insights

Calculated average age, highest fare, lowest fare, and total fare collection to understand dataset trends.
